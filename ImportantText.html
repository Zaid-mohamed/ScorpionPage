My Journey into Game Development

The Beginning: Zombie Fighter

My journey into game development began with a simple yet exciting project called Zombie Fighter. This zombie survival game was my first venture into creating interactive experiences.

In Zombie Fighter, players control a survivor trying to evade endless waves of pursuing zombies. The concept might be simple, but it taught me fundamental game development concepts like:

  • Player movement and controls
  • Enemy AI and pathfinding
  • Collision detection
  • Score tracking
  • Game loop management

What made this project special was how it combined HTML5 Canvas for graphics, JavaScript for game logic, and CSS for styling. The zombies spawn from different sides of the screen, creating an intense survival experience where quick thinking and strategic movement are key to survival.

The scoring system rewards players for surviving longer, with bonus points awarded when zombies leave the screen without catching the player. This creates an interesting risk-reward dynamic where players must decide between playing it safe or trying to maximize their score.

Key Features:

  • Smooth 360-degree player movement
  • Dynamic zombie spawning system
  • Progressive difficulty
  • Score tracking and display
  • Game over and reset functionality

While Zombie Fighter may seem simple compared to modern games, it represents an important milestone in my development journey. It taught me valuable lessons about game design, user experience, and the importance of starting small but thinking big.
